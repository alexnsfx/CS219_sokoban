niveau Pointeur sur le Niveau a remplir.
numero Numero du niveau a charger.

hauteurNiveau Pointeur sur l'entier qui contiendra la hauteur du niveau.
posJoueur Pointeur sur la Position du joueur, assignee des que le joueur est trouve dans le fichier.
pileCible Pointeur sur une pile de Cible, chaque Cible trouvee dans le niveau y est empilee.
Renvoie:
Renvoie UndefinedLevel si niveau est nul, UndefinedParameter si un des autres parametres est nul, FichierIntrouvable si le fichier de niveaux n'existe pas, AllocationMemoire si erreur d'allocation du Niveau, CaractereInconnu si un caractere inconnu est present dans le Niveau, NoError sinon.
